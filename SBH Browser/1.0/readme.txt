Sample Browser for HSP(SBH)

このブラウザは
・Microsoft WebView2
・hiro氏のmod_webview2コンポーネント(https://blog.goo.ne.jp/hiro239415/e/5041aec497e02427c2b123aef07977e3)
を用いております

===================================

このサンプルコードは、すぐにビルドすることも改良することも簡単になるようにしております。
多機能ブラウザを作りたい場合はInternetStrollerMemoriaのフォークをお勧めしますが、難解です。
こちらは機能が少ないですが、より直感的なサンプルです

===================================
また、以下の特徴があります

・本家InternetStrollerと同じくMITライセンスです

・本家InternetStrollerと異なり、まなPADやダークモード、UA変更、その他機能は実装しておりません

・本家InternetStrollerの1.4.0.0ベースですが、1.4.2.0の開発版の効率化も行っています

・本家InternetStrollerと*一切の互換性はありません*

・HSP開発者向けのもので、バイナリは配布しておりません(ユーザの方はInternetStrollerMemoriaをお使いください)

・HSP3.6でのx86ビルドを前提にしております

・ビルドする場合はSourse.hspを開いてビルドするだけです
